OpenStreetMap Vecto

OpenStreetMap Vecto is an ongoing effort to build a general-purpose configuration and stylesheet for Baremaps. It is used to generate vector tiles and to produce a Mapbox stylesheet inspired by OpenStreetMap Carto.
